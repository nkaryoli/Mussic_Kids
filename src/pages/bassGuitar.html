  2. How to Play a Note
    • Press a key corresponding to a note shown in the image.
    • Each key plays the sound of the note in standard bass tuning (EADG).

  4. Playing Tips
    • Play bass lines: Use different key combinations to create grooves and bass lines.
    • Experiment with rhythms: Try playing in different time signatures for a more dynamic performance.
    • Try chords: While the bass is usually played with single notes, you can press multiple keys to create harmonies.
    • Use sustain: Notes will ring out briefly, but you can press the key again to extend the sound.

With the Mouse

  1. Click to Play Notes
    • Click on the virtual fretboard or the corresponding string on the bass image.
    • Each click will produce the corresponding note, similar to plucking a real bass string.

Using Both Keyboard and Mouse

  1. Combine Inputs
    • Use the keyboard to play notes while clicking with the mouse to add effects.
    • Press a key and click at the same time to create harmonics.
    • Experiment by switching between keyboard and mouse for more control.
